
관리자 계정 관련 기능
교수자 계정 관리, 과정 관리, 과목 관리, 학생 관리, 성적 관리

교수자 계정 관리 기능
교수자 테이블(이름, 주번뒷자리)

과정 관리 기능
과정 테이블(과정명, 시작기간(연월일), 끝기간(연월일), 강의실 정보)
과목 여러개가 모여 과정 생성

과목 관리 기능
과목 테이블(과정명, 시작기간, 끝기간, 교재 명, 교수자 명 정보)

학생 관리 기능
학생 테이블(이름, 주번뒷자리)
중도 탈락 사실을 화면에서 확인할 수 있어야 한다.

성적 관리 기능
성적 테이블(출결, 실기, 필기)
비율은 교수가 결정
학생명, 과목명이 있으면 되지 않을까?

1. DB와 사용자 이용 화면 사이의 연결
2. 비밀번호를 암호화 처리하여 DB에 저장하는 기능
3. DB에 저장된 내용을 첨부파일로 사용자가 내려 받을 수 있도록 하는 기능


테이블

1. 관리자 테이블(ID, 비번)
2. 교수자 테이블(교수코드, 이름, 비번) - 주번 뒷자리
3. 학생 테이블(학생코드, 이름, 비번) - 주번 뒷자리
4. 중도 탈락 테이블(과정명, 학생명, 날짜)
5. 과정 테이블(과정코드, 과정명, 시작일, 종료일, 강의실)
6. 과목 테이블(과목코드, 과목명, 시작일, 종료일, 교재명, 교수코드, 진행여부, 과정코드)
7. 성적 테이블(과정코드, 과목코드, 학생코드, 출석, 실습, 필기, 총점, 등수)
